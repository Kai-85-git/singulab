# Singulab LLM ハッカソン

Singulab主催・スペースデータ兵頭氏出題の **LLMベース社会シミュレーションハッカソン** に向けた、アイデア整理と設計ドキュメントをまとめるリポジトリ。

- **提出締切**: 2026-05-07(木) 17:00
- **結果発表**: 2026-05-14 @ デジタルハリウッド大学(調整中)

## テーマ

LLMエージェントが住む小さな社会を構築し、**ルールでは書けない創発**が起きるかを観察する。

兵頭氏の **3階層フレームワーク** に沿って世界を設計:

| 階層 | 役割 |
| --- | --- |
| **1階:物** | 地形・建物・他エージェントの身体など物質的なもの |
| **2階:環境** | 温度・濃度・評判など目に見えないが影響する量 |
| **3階:物理・重力** | 世界を駆動する法則 |
| エージェント | LLM搭載。1〜3階を知覚 → 思考 → 行動 → 世界にフィードバック |
