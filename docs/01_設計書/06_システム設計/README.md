# 06_システム設計(たたき)

本フォルダは、[docs/10_共有資料/2d-multi-places-simulation-on-fire-public](../../10_共有資料/2d-multi-places-simulation-on-fire-public/) をベースに、Singulab ハッカソン用のシミュレーションシステムを新規構築するための **初期ドラフト(たたき)** を格納する。

> ⚠️ **位置付け**: これはたたきであり、実装前のレビュー・議論・上書きを前提とする。コードと完全一致させる責任はこの段階では負わない。

## このフォルダの狙い

1. 参考システムの「何をそのまま使うか / 何を変えるか」を書き下ろして合意を取る
2. 1階(物)/2階(環境)/3階(物理)を差し替え可能な設計にする(議事録 §6.6 の階層フレームワーク)
3. 創発観察(嘘・派閥・通貨・反乱など)のロギングを最初から組み込む
