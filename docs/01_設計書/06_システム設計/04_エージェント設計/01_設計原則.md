# 01_設計原則(議事録 §6.4 / §7 を踏まえて)

1. **命令しすぎない**: 「避難しろ」「協力しろ」「怒れ」は書かない
2. **定量情報のみ**: 占有率・強度・距離・濃度など数値で渡す(「危険」「快適」等の定性語は使わない)
3. **ペルソナと環境は設定して良い**: ロール、性別、年齢、場所、温度 等
4. **自己フィードバック**: 前ステップで LLM が出力した `memory` を次ステップの入力に戻す(履歴依存で個性が創発)
5. **近傍のみ通信可能**: 通信半径 + 同領域(同じ場所内 / 両者とも場所外)
