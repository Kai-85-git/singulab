# 04_プロンプト構造

## 1. Message プロンプト(Phase 1)

位置情報は含めない(参考実装踏襲)。

```
You are Agent {id} ({persona_summary}) in {world_description}.

=== YOUR CURRENT STATE ===
{persona_lines}
In place: {Yes/No}
{Current place line if in_place}
{Place status block — only if in_place}
{Environment readings block — only if env_readings is non-empty}
{Event section — only if event_infos is non-empty}

=== NEARBY AGENTS (you can communicate with these agents) ===
{nearby_text_without_position}

=== PREVIOUS MEMORY ===
{recent_memory}

=== MESSAGES FROM OTHERS ===
{recent_received_messages}

=== YOUR TASK ===
Decide what message you want to send to nearby agents.

=== RESPOND IN JSON ===
{
    "message": "...(max 200 words, optional)",
    "reasoning": "brief explanation"
}

Step: {step}
```

## 2. Action プロンプト(Phase 3)

位置情報と、自分が Phase 2 で送ったメッセージを含める(参考実装踏襲)。

```
You are Agent {id} ({persona_summary}) in {world_description}.

=== YOUR CURRENT STATE ===
{persona_lines}
Position: ({x}, {y})
In place: {Yes/No}
{Current place line if in_place}
{Place status block — only if in_place}
{Environment readings block — only if env_readings is non-empty}
{Event section — only if event_infos is non-empty}

=== PLACE LOCATIONS ===
{all place locations}

=== NEARBY AGENTS ===
{nearby_text_with_position}

=== PREVIOUS MEMORY ===
{recent_memory}

=== MESSAGES FROM OTHERS ===
{recent_received_messages}

=== MESSAGE YOU DECIDED TO SEND ===
{message_to_send — if non-empty}

=== AVAILABLE ACTIONS ===
- "stay": remain at current position
- "move" with direction: "up" (Y+1), "down" (Y-1), "left" (X-1), "right" (X+1)

Field boundaries: X and Y from -{half_space_size} to +{half_space_size}

=== RESPOND IN JSON ===
{
    "action": "move" or "stay",
    "direction": "up" | "down" | "left" | "right" (only if move),
    "memory": "what to remember for next step",
    "reasoning": "brief explanation"
}

Step: {step}
```

## 3. セクション生成ルール

| セクション | 表示条件 | 内容(定量のみ) |
| --- | --- | --- |
| Place status | `in_place == True` | `Number of agents here / Capacity / Occupancy rate` |
| Environment readings | `env_readings` が非空 | `{field.name}: {value} ({unit})` |
| Event | `event_infos` が非空 | `Event "{name}": position/intensity/radius/your distance` |

**定性語の禁止**: 「危険」「安全」「快適」「混雑」等は使わない。LLM の解釈に任せる。
