# 02_エージェント状態

| フィールド | 型 | 初期化 | 更新タイミング |
| --- | --- | --- | --- |
| `id` | int | config のインデックス | 固定 |
| `persona` | dict | config ルールに従い生成 | 固定(動的変化は後日検討) |
| `position` | (int, int) | 初期配置 | Phase 4(移動実行) |
| `in_place` | bool | 初期位置から算出 | Phase 2 と Phase 4 後 |
| `current_place` | str or None | 同上 | 同上 |
| `memory` | list[str] | `[]` | Phase 3 内部で `append`(LLM の `memory` 出力) |
| `received_messages` | list[dict] | `[]` | Phase 2 で他者から受信 |
| `stats` | dict | `{steps_in_place: 0, ...}` | 各ステップ終わり |

→ 参考実装の `Agent` とほぼ同じ。`gender` は `persona` dict に格納する。
