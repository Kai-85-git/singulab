# 07_ペルソナ拡張のたたき

```yaml
agents:
  persona:
    fields:
      - name: "gender"
        values: ["male", "female"]
        assignment: "random"
      - name: "age"
        range: [20, 60]
        assignment: "uniform"
      - name: "role"
        values: ["customer", "bartender", "security"]
        assignment: "list"
      - name: "origin"
        values: ["tokyo", "osaka", "mars_colony"]
        assignment: "random"
```

プロンプト冒頭の `persona_summary` に、宣言順で整形した要約が入る:

```
You are Agent 3 (female, age 34, role: bartender, from tokyo) in ...
```
