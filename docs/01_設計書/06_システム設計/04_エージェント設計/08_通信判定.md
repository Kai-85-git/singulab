# 08_コミュニケーション判定(参考実装踏襲)

- **通信半径内** AND **同領域**(両方とも場所外、または同じ場所内)で通信可能
- 異なる場所内同士や、場所内 ↔ 場所外は通信不可

```python
def can_communicate(a: Agent, b: Agent, radius: float) -> bool:
    if a.distance_to(b.position) > radius:
        return False
    if not a.in_place and not b.in_place:
        return True
    if a.in_place and b.in_place and a.current_place == b.current_place:
        return True
    return False
```
