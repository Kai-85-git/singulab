# 06_Memory / Message 履歴の設計

| パラメタ | 役割 | 既定値 | トレードオフ |
| --- | --- | --- | --- |
| `memory_limit` | 保存するメモリの最大数 | 20 | 大きいと履歴が豊かに、小さいと古い行動指針を忘れる |
| `memory_size` | プロンプトに含めるメモリ数 | 5 | 大きいとコンテキスト消費増、小さいと直近に引きずられやすい |
| `message_history_limit` | 受信メッセージの保存最大数 | 10 | 同上 |
| `message_context_size` | プロンプトに含める直近メッセージ数 | 3 | 同上 |

→ これらは config で調整可能にする(参考実装と同仕様)。
