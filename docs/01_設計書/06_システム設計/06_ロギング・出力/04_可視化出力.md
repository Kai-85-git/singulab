# 04_可視化出力

## 1. 参考実装踏襲

- エージェント色 = ペルソナ(例: 性別 → 青/赤)
- マーカー形状 = 場所内(★) / 場所外(●)
- イベント半径 = 破線円
- イベント強度 = カラーマップ(YlOrRd 等)+ カラーバー

## 2. 拡張

- **2階の環境量を背景ヒートマップとして重畳**(例: 飛沫濃度、フェロモン濃度)
- **通信エッジの描画(任意)**: Phase 2 で送受信された関係を線で結ぶ
- **指標ダッシュボード**: `statistics.png` に占有率以外の系列を追加

```
+------------------+-------------------+
| Frame(左大)     | Messages(右上)    |
|                  |                    |
|                  +--------------------+
|                  | Memory / Reasoning |
+------------------+--------------------+
```

→ `tools/viewer.html` は参考実装の HTML ビューアを踏襲し、`events.jsonl` と `metrics.jsonl` のパネルを追加する。
