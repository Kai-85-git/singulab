# 02_コンポーネント責務

| コンポーネント | 責務 | 参考実装の対応 |
| --- | --- | --- |
| **Simulation Orchestrator** | ステップ駆動、Phase 1〜4 の順序制御、イベント活性化、統計収集、ログ書き出し | `simulation.py` |
| **World(場・場所)** | 座標系、場所(place)ごとの `center / half_size / capacity`、場所内判定、通信可能判定 | `utils.py`(`PlaceConfig`, `is_position_in_place`)|
| **Agent** | LLM への推論依頼、Memory / Message / Action の生成と更新、近傍エージェント抽出 | `agent.py` |
| **Events(火事/他)** | 発生ステップ・中心位置・知覚半径・強度を持ち、エージェントに定量情報のみ提供 | `simulation.py` の `fire_states` 周辺 |
| **LLM Client** | Ollama API へのリクエスト、接続/モデル確認、生成オプション管理 | `ollama_client.py` |
| **Visualizer** | フレーム保存、統計プロット、カラーバー・通信半径・火事半径の描画 | `visualization.py` |
