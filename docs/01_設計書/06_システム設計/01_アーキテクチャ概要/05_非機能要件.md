# 05_非機能要件(たたき)

| 項目 | 指針 |
| --- | --- |
| 性能 | エージェント数 **5〜20** 体で 100 ステップを数分以内。LLM 呼び出しは逐次で可(並列化は後段) |
| 再現性 | `seed` を config に持たせる。LLM は `temperature` を明示。3〜10 回試行で統計比較 |
| 可観測性 | すべての LLM 入出力を `*.jsonl` で残す(後で解析可能にする) |
| モジュール性 | シナリオ追加時に `simulation.py` を大改修しなくても済むよう、World/Event/Agent の境界を明確に |
| 実行環境 | Python 3.10+ / Ollama(ローカル)+ 任意で API LLM。Windows / macOS で動かす |
