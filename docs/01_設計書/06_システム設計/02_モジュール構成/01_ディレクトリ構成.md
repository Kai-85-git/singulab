# 01_ディレクトリ構成案

```
singulab/
├── README.md
├── requirements.txt
├── config/
│   ├── base.yaml              # 共通デフォルト
│   ├── scenario_bar_fire.yaml # 参考実装と等価なシナリオ
│   ├── scenario_cells.yaml    # 細胞シナリオ
│   └── scenario_virus.yaml    # ウイルスシナリオ
├── src/
│   ├── __init__.py
│   ├── main.py                # CLI エントリポイント
│   ├── simulation.py          # Orchestrator
│   ├── world/
│   │   ├── __init__.py
│   │   ├── place.py           # Place / 場所の定義(1階)
│   │   ├── field.py           # 座標系・フィールド境界
│   │   └── environment.py     # 環境量(2階: 温度/濃度/評判 等)
│   ├── physics/               # 3階: 物理法則(拡散・重力・反応速度)
│   │   ├── __init__.py
│   │   ├── base.py            # 抽象 Physics
│   │   └── diffusion.py       # 拡散則(例)
│   ├── events/
│   │   ├── __init__.py
│   │   ├── base.py            # 抽象 Event
│   │   └── fire.py            # 火事イベント(参考実装由来)
│   ├── agent/
│   │   ├── __init__.py
│   │   ├── agent.py           # Agent 本体
│   │   ├── persona.py         # ペルソナ定義
│   │   ├── memory.py          # Memory / Message 履歴
│   │   └── prompt.py          # プロンプト組み立て(Message / Action)
│   ├── llm/
│   │   ├── __init__.py
│   │   ├── base.py            # LLMClient 抽象
│   │   ├── ollama.py          # Ollama 実装
│   │   └── anthropic.py       # Claude 実装(任意)
│   ├── logging/
│   │   ├── __init__.py
│   │   ├── jsonl.py           # messages / memory_reasoning / events のロガー
│   │   └── metrics.py         # 創発指標の算出
│   └── viz/
│       ├── __init__.py
│       ├── visualizer.py      # matplotlib での描画
│       └── video.py           # 動画生成(FFmpeg)
├── tools/
│   ├── viewer.html            # ブラウザビューア(参考実装由来)
│   └── generate_video.py      # 動画生成 CLI
├── output/                    # 実行時生成(gitignore)
│   ├── frame_XXXX.png
│   ├── messages.jsonl
│   ├── memory_reasoning.jsonl
│   ├── events.jsonl
│   └── statistics.png
└── tests/
    ├── test_world.py
    ├── test_agent.py
    └── test_events.py
```
