# 04_プラグイン追加のテンプレート

## 1. 新しい Event を追加する手順

1. `src/events/<new_event>.py` に `class NewEvent(Event)` を作る
2. `activate / is_active / perceived_by` を実装
3. `src/events/__init__.py` に `EVENT_REGISTRY["new_event"] = NewEvent` を登録
4. `config.events` で `type: "new_event"` が使えるようになる

## 2. 新しい Physics を追加する手順

1. `src/physics/<new_law>.py` に `class NewLaw(Physics)` を作る
2. `step(env, dt)` を実装
3. レジストリに登録
4. `config.physics.laws` から呼び出し可能に

## 3. 新しい LLM Provider を追加する手順

1. `src/llm/<provider>.py` に `class NewProvider(LLMClient)` を作る
2. `generate / check_connection / list_models / check_model_exists` を実装
3. レジストリに登録
4. `config.llm.provider: "<provider>"` で切り替え可能に
