# 03_シナリオ別の拡張ポイント

## 1. 細胞(議事録 §8.3)

| 階層 | 必要な追加 |
| --- | --- |
| 1階 | 細胞の身体(円形)、ECM、血管壁 |
| 2階 | `glucose`, `oxygen`, `pH`, `cytokine` の各フィールド |
| 3階 | 拡散、反応速度、浸透圧(ガウス拡散だけで最小実装可) |
| Agent | ペルソナに `cell_type`(幹細胞/血管細胞/etc)追加、役割選択を LLM に任せる |
| Event | 損傷・感染などの外乱 |

**期待される観測**:
- 局所のサイトカインに応じた分化選択
- アポトーシスの自発発生(「過密を解消するため自ら消える」)
- 血管の自己組織化(栄養勾配へのフィードバック)

## 2. ウイルス × 人間

| 階層 | 必要な追加 |
| --- | --- |
| 1階 | 人間の身体、ウイルス粒子、マスク、建物 |
| 2階 | `droplet_concentration`, `immunity_level`, `reputation`(差別) |
| 3階 | 飛沫拡散、感染確率、免疫減衰 |
| Agent | 人間とウイルス粒子の 2 タイプ、`health_state` フィールド |
| Event | アウトブレイク(初期感染)、ワクチン配給 |

**期待される観測**:
- 弱毒化戦略への自発的シフト
- 感染者への差別 vs 連帯の分岐
- 外出抑制 → 感染曲線の平坦化

## 3. 月面独立運動(議事録 §8.1)

| 階層 | 必要な追加 |
| --- | --- |
| 1階 | 居住モジュール、物資、通信機 |
| 2階 | 地球からの指令(情報フィールド)、物資残量、経済指標 |
| 3階 | エネルギー制約、通信遅延 |
| Agent | `origin`(earth/moon) を持つペルソナ、世代別価値観 |
| Event | 通信遮断、物資補給遅延 |

**期待される観測**:
- 地球への反抗メッセージの自発発生
- 月面独自通貨の提案
- 派閥形成(親地球派 vs 独立派)
