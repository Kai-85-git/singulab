# 04_環境4象限と比較方式

## 1. 4 象限構造

「都心 vs 地方」× 「大企業 vs スタートアップ」の組み合わせ:

|  | 地方 | 都心 |
| --- | --- | --- |
| 大企業 | □ | □ |
| スタートアップ | □ | □ |

## 2. 進め方

- **まず片方の軸(例: 都心)で 2 つ(大企業 / スタートアップ)を回す**
- 時間に余裕があれば、もう片方(地方)も実施
- エージェントのプロンプトは「**あなたは◯◯(場所)の△△(規模)にいる□□歳です**」程度に留める

## 3. 比較の実装方式(3 通り、実装フェーズで決定)

1. 2 つの箱を**同時並行**でステップを回す
2. **個別**で回して比較する
3. 箱を**くっつけて**比較する

→ **基本方針は箱 2 つの並列比較方式**。具体形は実装フェーズで決定。
