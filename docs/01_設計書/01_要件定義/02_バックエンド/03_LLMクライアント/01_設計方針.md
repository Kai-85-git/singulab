# 01_設計方針

- **OpenAI 互換 API を常に使う** (`POST /v1/chat/completions`)
  - Ollama / llama-server / vLLM / Anthropic API(ラッパー経由) をすべて同じ呼び出し方で扱う
- **`base_url` を env で切替可能に** (`OLLAMA_BASE_URL` / `LLM_BASE_URL`)
- **async 前提**: 100 エージェントの並列実行を支えるため
- **厳密な出力検証**: pydantic で LLM レスポンスを JSON schema にバインド
- **リトライ + 指数バックオフ**: ノート PC の過渡的な OOM や Ollama モデル再ロード時間を吸収
