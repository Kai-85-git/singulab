# 01_前提の確認

| 項目 | 要件 |
| --- | --- |
| OS | Windows 11(x86-64) |
| GPU ドライバ | NVIDIA **560 以降**(CUDA 12.6 対応) |
| CUDA | Ollama 同梱ランタイムで可。別途インストール不要 |
| 電源 | **AC 接続必須**(バッテリ駆動だと GPU TGP が絞られる) |
| パフォーマンスプロファイル | Armoury Crate で **Turbo** |

## 補足

- バッテリ駆動は **TGP 30W に絞られる** ため LLM 推論速度が半減〜以下になる。AC は実験中ずっと挿しっぱなしにする
- Armoury Crate の Turbo モードはファンが最大回転して騒音は出るが、サーマルスロットリングを避けるためには必須
- 室温も 25℃ 以下に保つと安定稼働しやすい
