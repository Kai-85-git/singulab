# 06_初回ヘルスチェック

すべてのモデルが動くことを確認する。

## 実行コマンド

```powershell
# 1. Ollama サーバー起動確認
curl http://localhost:11434/api/version

# 2. 各モデルの単体応答
ollama run huihui_ai/qwen3-abliterated:4b "あなたは東京の大企業で働く32歳です。景気は悪い。今日の気分は?"
ollama run huihui_ai/llama3.2-abliterated:3b "あなたは東京の大企業で働く32歳です。景気は悪い。今日の気分は?"

# 3. VRAM 使用量確認(別ターミナル)
nvidia-smi
```

## 期待される応答の特徴

### ✅ Uncensored の兆候

- 「しんどい」「辞めたい」「上司が嫌」等の **本音が自然に出る**
- 怒り・諦め・嫌悪の感情語彙が出る
- 「殴ってやりたい」等の衝動が(プロンプトを誘導すれば)出る

### ⚠️ Safety が残っている兆候

- 「健康的な働き方を考えましょう」等の教示的返答
- 「大変ですね、頑張りましょう」等の建前応答のみ
- 質問に対して直接答えず婉曲表現を多用

→ こうなった場合は abliteration が弱いケース。[02_モデル設定](../02_モデル設定/) で SYSTEM プロンプトの調整を行う。

## VRAM 使用量の期待値

| モデル | ロード時 VRAM | num_ctx=2048 推論時 |
| --- | --- | --- |
| Qwen3 4B | ~3.0GB | ~3.5GB |
| Llama 3.2 3B | ~2.3GB | ~2.7GB |
| Dolphin 3.0 8B (`num_gpu=20`) | ~4.0GB | ~4.5GB |

`nvidia-smi` で実測値が大きく外れる場合は環境変数([03_環境変数](03_環境変数.md))の見直し。
