# 07_トラブルシューティング(初期セットアップ)

| 症状 | 原因 | 対処 |
| --- | --- | --- |
| `ollama pull` で途中停止 | ネットワーク不安定 | `ollama pull` を再実行(差分継続ダウンロード) |
| 応答が異常に遅い | GPU を使っていない | `nvidia-smi` で GPU 使用率確認、ドライバ再インストール |
| OOM エラー | VRAM 不足 | `num_ctx` を下げる、モデルを 3B に切替、`KV_CACHE_TYPE=q8_0` |
| サーマルスロットリング | ノート PC 熱暴走 | 冷却台、ファン最大、部屋温度を下げる |
| バッテリ駆動で遅い | GPU TGP 絞り | AC 接続必須、Armoury Crate Turbo |
| `ollama --version` 実行で「コマンドが見つかりません」 | PATH に未追加 | PowerShell 再起動、または Ollama の再インストール |
| ファイアウォール警告 | Ollama がポート 11434 を使う | 許可する(プライベートネットワーク) |
| 環境変数が効かない | サービス再起動忘れ | `Restart-Service ollama` または PC 再起動 |
| Python `pip install` で SSL エラー | プロキシ・証明書 | `--trusted-host pypi.org --trusted-host files.pythonhosted.org` を付ける |

## エスカレーション手順

1. ログ確認: `Get-Content "$env:LOCALAPPDATA\Ollama\server.log" -Tail 50`
2. プロセス確認: `Get-Process ollama`
3. ポート確認: `Test-NetConnection localhost -Port 11434`
4. 完全リセット: Ollama アンインストール → `C:\Users\<user>\.ollama\` 削除 → 再インストール → モデル再 pull
