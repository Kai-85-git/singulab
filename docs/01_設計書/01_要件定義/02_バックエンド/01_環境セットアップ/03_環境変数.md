# 03_環境変数

## 設定する変数

| 変数 | 値 | 用途 |
| --- | --- | --- |
| `OLLAMA_NUM_PARALLEL` | `8` | 同時処理リクエスト数(3B モデル時。4B なら 6、8B なら 1) |
| `OLLAMA_FLASH_ATTENTION` | `1` | Flash Attention 有効化(Ampere でも効く、必須) |
| `OLLAMA_KV_CACHE_TYPE` | `q8_0` | KV キャッシュを 8bit 量子化して VRAM 節約 |
| `OLLAMA_KEEP_ALIVE` | `10m` | モデルをメモリに保持する時間(切替時の再ロード防止) |
| `OLLAMA_MAX_LOADED_MODELS` | `1` | 同時にロードするモデル数(VRAM 逼迫回避) |

## セッション単位で設定

```powershell
$env:OLLAMA_NUM_PARALLEL = "8"
$env:OLLAMA_FLASH_ATTENTION = "1"
$env:OLLAMA_KV_CACHE_TYPE = "q8_0"
$env:OLLAMA_KEEP_ALIVE = "10m"
$env:OLLAMA_MAX_LOADED_MODELS = "1"
```

設定後は **Ollama サービスを再起動** または PC 再起動が必要。

## 永続化(再起動後も有効)

```powershell
[Environment]::SetEnvironmentVariable("OLLAMA_NUM_PARALLEL", "8", "User")
[Environment]::SetEnvironmentVariable("OLLAMA_FLASH_ATTENTION", "1", "User")
[Environment]::SetEnvironmentVariable("OLLAMA_KV_CACHE_TYPE", "q8_0", "User")
[Environment]::SetEnvironmentVariable("OLLAMA_KEEP_ALIVE", "10m", "User")
[Environment]::SetEnvironmentVariable("OLLAMA_MAX_LOADED_MODELS", "1", "User")
```

## モデルサイズに応じた `OLLAMA_NUM_PARALLEL` の指針

| 使用モデル | 推奨並列度 |
| --- | --- |
| Llama 3.2 3B | **8〜12** |
| Qwen3 4B | **6〜8** |
| Dolphin 3.0 8B(CPU オフロード) | **1** |
