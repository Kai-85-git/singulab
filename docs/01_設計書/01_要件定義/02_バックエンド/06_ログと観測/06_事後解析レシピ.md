## 1. pandas で読み込む

```python
import pandas as pd
df = pd.read_json("logs/runs/exp_20260425_001/llm_io.jsonl", lines=True)
print(df.groupby("agent_id").response_parsed.apply(
    lambda s: sum(1 for r in s if r["utterance"] == "")
))  # エージェント別沈黙数
```

## 2. 発言権の時系列推移

```python
by_step = df.groupby(["step", "agent_id"]).size().unstack(fill_value=0)
gini_timeseries = by_step.apply(lambda row: gini(row.values), axis=1)
gini_timeseries.plot()  # 大集団ほど Gini が上がる、等が見える
```
