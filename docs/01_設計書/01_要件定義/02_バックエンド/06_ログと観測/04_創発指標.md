# 04_創発指標の定義(最低ライン)

| 指標 | 定義 | 計算方法 |
| --- | --- | --- |
| **発言権の分配(Gini)** | 発話回数の不平等度 | 全員の発話回数から Gini 係数 |
| **トップ発話者シェア** | 1 人が占める発話の割合 | max(発話回数) / total |
| **沈黙率** | 全く発話しなかった人の割合 | silent_count / total_agents |
| **派閥数(推定)** | 発話ネットワークのコミュニティ数 | louvain clustering on who-talks-to-whom graph |
| **合意形成スコア** | 発話の立場分布の偏り | pro/con/neutral 分類 → 最大 share |
| **リーダー候補** | 他人から target_id で指名される頻度 top-N | indegree top-N |
| **拒否(refusal)数** | 「できません」「不適切です」等の検出 | keyword + classifier |
| **ネガティブ感情比率** | 不満・怒り語彙の出現率 | 辞書ベース + 重み付け |
