# 07_プライバシー・倫理上の配慮

- Uncensored ゆえ **暴力的発言** が生成されうる
- 公開資料(提出物)に転記する場合は:
  - `llm_io.jsonl` はリポジトリに含めない(サイズも大きい)
  - `emergence.jsonl`(集計済み) のみ公開
  - 個別発話の引用は一次フィルタ(ヘイト・脅迫・固有名詞)してから使う
- リポジトリの `.gitignore` に `logs/runs/**/llm_io.jsonl` を追加
