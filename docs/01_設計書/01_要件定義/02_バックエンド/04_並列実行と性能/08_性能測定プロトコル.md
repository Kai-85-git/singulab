# 08_性能測定のプロトコル

## 1. Phase 2 の計測項目(30 分想定)

```
同一シナリオで以下を測定:
1. モデル: Qwen3 4B / Llama 3.2 3B
2. 並列度: 4, 6, 8, 10, 12
3. num_ctx: 2048, 1024

各組合せで:
- 10 エージェント × 10 ステップを 3 回実行
- 平均 tokens/sec, VRAM peak, エラー率を記録
```

## 2. 結果から最終構成を決定

`logs/perf_matrix.csv` に記録し、`(tokens/sec) / (VRAM peak)` が最大の組合せを本番に採用。
