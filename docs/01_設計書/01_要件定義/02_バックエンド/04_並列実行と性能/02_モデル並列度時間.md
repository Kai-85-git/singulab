# 02_モデル × 並列度 × 時間

| 構成 | スループット(合計) | 完走時間 |
| --- | --- | --- |
| 3B + 並列 12 | ~350-450 tok/s | **30-40 分** ✅ 最速 |
| 3B + 並列 8 | ~250-350 tok/s | 45-55 分 ✅ |
| 4B + 並列 8 | ~220-280 tok/s | 50-60 分 ✅ |
| 4B + 並列 4 | ~140-180 tok/s | 80-100 分 |
| 8B + CPU オフロード + 並列 1 | ~10 tok/s | 約 22 時間 ❌ |

→ **100 人本番は 3B + 並列 8-12** が現実解。
