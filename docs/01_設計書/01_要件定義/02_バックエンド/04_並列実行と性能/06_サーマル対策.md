# 06_サーマル対策(ノート PC)

長時間稼働での throttle を避ける:

| 対策 | 効果 |
| --- | --- |
| 冷却台を使用 | CPU/GPU 温度 -5〜10℃ |
| 筐体背面を 2cm ほど浮かす | 底面吸気改善 |
| Armoury Crate を **Turbo** に固定 | ファン最大速度 |
| 室温を 25℃ 以下に | 熱余裕 |
| **1 時間ごとに 5 分休憩** | 長期安定性 |

温度が 90℃ を超え続けるとスロットリングで tok/s が 30-50% 落ちる。**80℃ 以下を維持** するのが理想。
