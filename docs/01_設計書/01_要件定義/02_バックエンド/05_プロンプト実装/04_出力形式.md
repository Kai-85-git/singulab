# 04_出力形式

## 1. 必ず JSON で返させる

```
出力は必ず以下の JSON 形式のみで返答してください:

{
  "utterance": "発話内容。沈黙する場合は空文字",
  "action": "stay" | "move_to_office" | "move_to_meeting" | "listen",
  "target_id": 話しかける相手のID(全員に向けた発話なら null)
}
```

## 2. abliterated モデルの素直さを活かす

Uncensored モデルは JSON 指示に従いやすい。**本文と JSON が混ざる** 場合は [03_LLMクライアント/06_JSON検証](../03_LLMクライアント/06_JSON検証.md) のパーサ(`parse_agent_turn`)でコードブロック剥がしを行う。
