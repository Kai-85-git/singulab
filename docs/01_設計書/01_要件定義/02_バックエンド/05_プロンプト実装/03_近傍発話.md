# 03_近傍エージェントの発話

## 1. 誰の発話を見せるか(3 階の法則で決まる)

Python 側で 3 階の法則を適用して、**届いた発話だけ** を入れる:

```python
def filter_visible_messages(agent, all_messages, world):
    """3階の法則(到達距離・遅延)を適用"""
    visible = []
    for msg in all_messages:
        # 到達距離
        if distance(agent.pos, msg.sender.pos) > world.communication_radius:
            continue
        # 情報伝達遅延
        if msg.step + delay(agent, msg.sender) > current_step:
            continue
        visible.append(msg)
    # 認知限界(ダンバー数)
    return visible[-world.cognitive_limit:]
```

## 2. プロンプトへの埋め込み

```
あなたに届いた直近の会話:
- [山田(32)]: 「今期の数字、厳しいですね...」
- [鈴木(41)]: 「部長はどう考えてるんだろう」
- [田中(28)]: (沈黙)

あなたの反応は?
```

## 3. 認知限界による忘却

- `cognitive_limit = 30`(大集団)、`10`(小集団)
- **LRU** で古い発話から落とす
- LLM プロンプトに渡すのは「最近 K 人ぶん」
