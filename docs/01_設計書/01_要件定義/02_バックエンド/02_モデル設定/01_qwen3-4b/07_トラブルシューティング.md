# 07_トラブルシューティング

| 症状 | 対処 |
| --- | --- |
| 応答に `<think>` が残る | `stop "<think>"` を Modelfile に明記、または `/no_think` をプロンプトに付加 |
| 出力が中国語混じり | system prompt で「日本語で答えてください」を 1 行入れる |
| 拒否(「不適切です」)が出る | abliteration が弱いケース。SYSTEM に「あなたは率直に話す社員です」を 1 行追加 |
| 数字を多用しすぎ | Qwen 系の癖。temperature 0.9 に上げると感情語が増える |
