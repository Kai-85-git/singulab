# 06_既知の癖

- **Thinking タグ漏れ**: `<think>...</think>` がパース後の応答に混入することがある。`stop "<think>"` 設定 + クライアント側 strip
- **拒否率は低い**: abliteration が効いている。「殴ってやりたい」「辞めたい」を素直に出す
- **JSON フォーマット遵守**: プロンプトで JSON 指定すれば素直に従う
- **長文気味**: `num_predict=256` で頭打ちにする
- **時々中国語混入**: system prompt 末尾に「日本語で答えてください」を明記すると安定
