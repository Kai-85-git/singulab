# 04_推奨パラメータ(本モデル特有)

| パラメータ | 値 | 理由 |
| --- | --- | --- |
| `temperature` | **0.8** | Qwen3 は出力が固めなので少し高め |
| `num_ctx` | 2048 | 32K まで可だが VRAM 節約 |
| `num_predict` | 256 | 短文発話 |
| `stop` | `"<think>"` | thinking モード抑制 |

## thinking モード OFF の代替方法

Modelfile に書かなくても、プロンプト末尾に `/no_think` を付ければ抑制可能:

```python
user_prompt = f"{messages_text}\n\nあなたの反応は?\n/no_think"
```
