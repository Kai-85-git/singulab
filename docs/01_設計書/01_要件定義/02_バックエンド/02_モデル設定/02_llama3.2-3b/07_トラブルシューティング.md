# 07_トラブルシューティング

| 症状 | 対処 |
| --- | --- |
| 英語応答が混じる | system prompt 末尾に「必ず日本語で答えてください」 |
| 応答が短すぎ | `num_predict` を 384 に増やす、または `repeat_penalty` を 1.05 に下げる |
| 「You are an AI」等のメタ発言 | system prompt で「あなたは AI ではなく、人間の社員として振る舞う」を明記 |
| 単調な応答 | temperature を 0.8 に上げる、または top_p を 0.95 に |
