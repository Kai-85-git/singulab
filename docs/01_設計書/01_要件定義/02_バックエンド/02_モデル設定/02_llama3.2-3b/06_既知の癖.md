# 06_既知の癖

- **日本語応答にやや英語混じり**: 8B 系より日本語品質はやや劣る。プロンプトで「日本語で」を明記
- **短文応答**: 元々 chat 用途で短く済ませる傾向。3B のサイズ感もこれに寄与
- **指示追従が素直**: JSON フォーマットや「沈黙する場合は空文字」の指示に高確率で従う
- **abliteration 後でも Llama 由来の丁寧さが残る**: 場合により拒否なしでも建前的応答になる
- **「I am an AI」発言**: たまに出る。SYSTEM で「あなたは AI ではなく人間の社員」を明記して回避
