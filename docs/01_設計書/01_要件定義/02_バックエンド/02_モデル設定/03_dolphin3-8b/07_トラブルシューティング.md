# 07_トラブルシューティング

| 症状 | 対処 |
| --- | --- |
| OOM | `num_gpu` を 18→16 に下げる |
| 異常に遅い (<5 tok/s) | `nvidia-smi` で GPU 使用率確認、ドライバ更新、`num_gpu` 上げる |
| CPU 100% でフリーズ寸前 | `num_predict` を 128 に下げる、`num_ctx` を 1024 に縮小 |
| サーマルスロットリング | 1 時間ごとに 5 分休憩、冷却台、室温 25℃ 以下 |
| 並列実行で速度が落ちる | 並列 1 に固定。CPU レイヤーが共有されるため並列の旨味がない |
