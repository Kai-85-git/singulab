# 04_推奨パラメータ(本モデル特有)

| パラメータ | 値 | 理由 |
| --- | --- | --- |
| `temperature` | 0.7 | Dolphin はやや饒舌、0.7 が安定 |
| `num_ctx` | 2048 | 節約 |
| `num_predict` | **192**(他より短め) | 8B の遅さを補う |
| `num_gpu` | 20 | CPU+GPU バランス |
