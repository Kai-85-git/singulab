# 04_非機能要件

| 項目 | 指針 |
| --- | --- |
| 性能 | 小集団 2〜10 人 + 大集団 100 人で 100 ステップを現実的時間内に完走 |
| 計算資源 | 人数 = LLM 数。PC がパンクしない範囲で運用(100 人が当面の上限) |
| 再現性 | シード・パラメータを config 化。同一条件の反復試行が可能 |
| 可観測性 | 全エージェントの LLM 入出力・行動ログを保存(嘘・派閥等の事後解析用) |
| モジュール性 | 1階/2階/3階を差し替え可能な設計(参照: [06_システム設計](../../06_システム設計/)) |
| 実行環境 | Ollama(ローカル)を基本。Windows / macOS で動かす |
