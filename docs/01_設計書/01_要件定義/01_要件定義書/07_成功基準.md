# 07_成功基準

- [ ] 小集団・大集団で **意思決定プロセスに観測可能な差異** が現れること
- [ ] 「あえて設定しない」項目(合意形成・発言権)が **勝手に創発** し、ログから抽出できること
- [ ] 100 ステップが現実的時間内で完走すること
- [ ] 再現性のある結果が取れ、条件を振った比較実験が回せること
- [ ] 「何人から大集団化するか」の閾値が観察できること(可変人数設計の検証)
