# 03_用語定義

| 用語 | 定義 |
| --- | --- |
| 小集団 | 2〜10人の可変人数集団。スタートアップに相当 |
| 大集団 | 100人規模の集団。大企業に相当 |
| エージェント | LLM で駆動する個々の参加者。人数=LLM数 |
| 3階層モデル | 1階(物) / 2階(環境) / 3階(世界の法則) の設計フレームワーク([キックオフ議事録 §6.6](../../../99_その他/2026-04/2026-04-22_LLMハッカソン_キックオフ議事録.md)) |
| 創発 | 個々の設定から導けない、集団レベルで立ち上がる現象 |
| 「あえて設定しない」 | 創発観察のため意図的に明示しない設計方針。振り返りの説明変数 |
| Uncensored モデル | 安全制約を緩めた脱制約 LLM。衝動的な人間らしさを引き出す目的で採用 |
