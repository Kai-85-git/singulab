# 06_制約条件

- **計算資源制約**: 人数 = LLM 数のため 1000 人規模は不可。100 人が現実的な上限
- **「あえて設定しない」原則**: 役職・合意形成ルール・発言権分配を事前に定めない
- **人間キャパシティの尊重**: 3 階の法則は記憶・認知など人間の限界の範囲内に収める
- **プロンプトは最小限**: 「◯◯(場所)の△△(規模)にいる□□歳」程度。命令を増やすほど創発が死ぬ
- **触れてはいけないファイル**: 粉川氏作成の「注意フォルダ」内のファイルは変更不可(リスト化予定)
