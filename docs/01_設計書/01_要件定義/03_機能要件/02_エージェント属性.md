# 02_エージェント属性

## 1. 基本属性

| 属性 | 基本設定 | 備考 |
| --- | --- | --- |
| 男女比 | **8:2**(男性多め) | 時間があれば 7:3 も比較 |
| 国籍 | 日本人ベース、**10人中1人程度外国人** | 外国人の人数閾値で創発が変わるか観察 |
| 年齢層 | 年齢のみ付与 | 役職等は付与しない |
| 役職 | **設定しない** | 自然発生を観察するため |

## 2. 設計ポリシー(兵頭氏アドバイス準拠)

- **指示を書きすぎない**。プロンプトで命令するほど創発が死ぬ
- 役職・階層は「勝手に握手が決まっていく」様を観察したいのであえて定義しない
- LLM の国籍差(作られた国による性格差)も観察対象として扱う
