# 05_実行パラメータ

## 1. パラメータ一覧

| 項目 | 設定 | 備考 |
| --- | --- | --- |
| ステップ数 | **100 ステップ**程度 | これ以上は時間がかかるため |
| ステップ時間計測 | 10 ステップを回して 10 倍で推定 | 総時間の事前見積もり |
| LLM モデル | **Uncensored 系(脱制約モデル)** | 詳細は [02_LLMモデル/2026-04-23_候補モデル一覧](../../02_LLMモデル/2026-04-23_候補モデル一覧.md) |

## 2. Uncensored モデル採用の思想(兵頭氏との議論由来)

- 通常の LLM は安全制約により「殴ってやろう」「こいつをクビにしたい」等の**人間らしい衝動**を出さない
- 賢い LLM ほど「頭のいい会話」になりがちで、純粋で衝動的な人間の側面が見えない
- **リミッター解除状態のモデル**を使うことで、より人間らしい行動データを取得
- Ollama 等で動かせるベースで選定
