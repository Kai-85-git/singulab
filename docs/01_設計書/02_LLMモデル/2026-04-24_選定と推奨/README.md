## 🎯 採用決定 (2026-04-25)

**Uncensored (abliterated) 系モデルを採用**と確定。

### 根拠
- 議事録 §7: 「純粋で衝動的な人間の側面を見るにはリミッター解除状態のモデル」(兵頭氏アドバイス)
- 本書 §02_設計哲学: 3-4B の abliterated が「実人間の認知能力 + 低拒否率」で観察したい創発を最大化する
- 議事録の観察項目 7 つ中 4 つ(忖度・諦め・デマ・フリーライド)が Safety-aligned では物理的に見えない

### 採用モデル
- ★★★ **Qwen3 4B abliterated** (🇨🇳 メイン・小集団/大集団兼用)
- ★★★ **Llama 3.2 3B abliterated** (🇺🇸 100 人大集団の本命)
- ★ (stretch) **Dolphin 3.0 abliterated 8B** (🇺🇸 品質検証用・小集団のみ)

### 除外(今回採用しない)
- 🚫 日本製モデル全般(公式 abliterated 版なし、Uncensored 要件不適合)
- 🚫 70B+ 大型モデル(頭良すぎ問題で創発が死ぬ / VRAM 不足)
- 🚫 Safety-aligned 通常モデル(本選定の対照実験として使う余地はあるが、本線ではない)

### 次のアクション
→ [10_検証計画](10_検証計画.md) の **フェーズ 1 スモークテスト(20 分)** を実行。
